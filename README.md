# fours_for_ten

# Input
- four numbers
- e.g.
    - 1234
    - 5555

# Output
- List of formula that results in 10
- e.g. 
    ```
    *** List(1234) ***
    0:1*2*3+4=10
    1:1+2+3+4=10
    sum:2
    ```
